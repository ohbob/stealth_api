# {title}

Attack mobile with id "ObjdID"
If you are not in War mode, the client toggle it before the attack.



Pascal

```pascal
procedure Attack(ObjdID:Cardinal)
```

Python

```python
Attack(ObjdID)
```

Example 1:

```pascal
if (WarTargetID <> Enemy) then
  attack(Enemy);
```

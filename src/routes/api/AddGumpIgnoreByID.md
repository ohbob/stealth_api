# {title}

Add "ID" to GumpIgnore list, so gumps with this ID will be ignored whencoming from server.



Pascal

```pascal
procedure AddGumpIgnoreByID(ID : Cardinal);
```



Python

```python
AddGumpIgnoreByID(ID)
```

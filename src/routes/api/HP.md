# {title}

Returns character property *HP* (life)

**Pascal Syntax:**

```pascal
function HP : Integer;
```
Pascal Example:
```pascal
while HP < 50 do
begin
  WaitTargetSelf;
  UseObject(bandage_type);
end;
```

**Python Syntax:**
```python
def HP(): --> int
```

# {title}

Cast spell with *SpellName*.
List of default spell you can find lower (keep in mind, that it can differ on some shards)

Pascal

```pascal
procedure Cast(SpellName : String);
```



Python

```python
Cast(SpellName)
```



Example Python

```python
Cast("Heal")
```

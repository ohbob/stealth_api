# {title}

Return skill lock state. 

Possible values: 0 - skill will increase, 1 - skill will decrease, 2 - skill locked.

**Pascal Syntax:**

```pascal
function GetSkillLockState(SkillName : String) : ShortInt;
```	

**Python Syntax:**
```python
def GetSkillLockState(SkillName): --> byte
```

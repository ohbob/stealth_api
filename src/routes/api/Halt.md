# {title}

Immendiately stops the current running internal script (PS or Lape).



Pascal

```pascal
procedure Halt;

```




Example Pascal

```pascal
program test;
begin
if Dead then 
begin 
    AddToSystemJournal('Char dead!');
    Disconnect;
    Halt;
end;
```

# {title}


Add "Serial" to GumpIgnore list, so gumps with this Serial will be ignored when coming from server.



Pascal

```pascal
procedure AddGumpIgnoreBySerial(Serial : Cardinal);
```



Python

```python
AddGumpIgnoreBySerial(Serial)
```
